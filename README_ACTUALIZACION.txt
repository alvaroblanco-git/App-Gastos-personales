## Cómo actualizar tu repo GitHub Pages
1. Descarga este ZIP.
2. Descomprímelo.
3. En tu repositorio de GitHub, reemplaza:
   - index.html
   - manifest.json
   - service-worker.js
4. Haz commit de los cambios.
5. Espera 1-2 minutos y recarga tu web.
